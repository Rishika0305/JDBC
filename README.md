# JDBC CRUD Operations Using Java and MySQL

## 📌 Project Overview

This project demonstrates the implementation of **CRUD (Create, Read, Update, Delete)** operations using **Java**, **JDBC**, and **MySQL**. It showcases how Java Database Connectivity (JDBC) enables interaction with a MySQL database for performing employee record management.

---

## 🛠 Technologies Used

* Java
* JDBC
* MySQL
* XAMPP
* Eclipse IDE

---

## ✨ Project Features

* Insert Employee Record
* Display Employee Records
* Update Employee Details
* Delete Employee Record
* Batch Insertion
* Database Connectivity using JDBC

---

## ⚙ Software Requirements

* Java JDK
* Eclipse IDE
* XAMPP Server
* MySQL JDBC Connector

---

## ▶️ How to Run

1. Install Java JDK.
2. Install Eclipse IDE.
3. Install XAMPP.
4. Start Apache and MySQL from the XAMPP Control Panel.
5. Open phpMyAdmin.
6. Create a database named `jdbc`.
7. Create the `employee` table.
8. Add the MySQL JDBC Connector JAR file to the project libraries.
9. Import the project into Eclipse.
10. Run the Java files as a Java Application.
