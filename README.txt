JDBC CRUD Operations Using Java and MySQL

Technologies Used:
- Java
- JDBC
- MySQL
- XAMPP
- Eclipse IDE

Project Features:
- Insert Employee Record
- Display Employee Records
- Update Employee Details
- Delete Employee Record
- Batch Insertion
- Database Connectivity using JDBC

Software Requirements:
- Java JDK
- Eclipse IDE
- XAMPP Server
- MySQL JDBC Connector

How to Run the Project:
1. Install Java JDK
2. Install Eclipse IDE
3. Install XAMPP
4. Start Apache and MySQL from XAMPP Control Panel
5. Open phpMyAdmin
6. Create database 'jdbc'
7. Create table 'employee'
8. Add MySQL JDBC Connector JAR to project libraries
9. Import project into Eclipse
10. Run Java files as Java Application
